# Wei · 个人主页 + 万能工具箱（增强版）

✨ 一个简洁现代的个人主页与 **18 合 1 全能前端工具箱**，全静态、零依赖，部署于 GitHub Pages。  
工具箱采用**卡片式折叠交互**，点击标题即展开工具，界面清爽，每个工具均经过实用性深度优化。

## 🌟 特性

- **现代个人主页**：动态时间问候、深色/浅色主题、技术技能展示、快速链接（GitHub、仓库、工具箱入口）
- **增强版工具箱**：18 个实用工具，涵盖计算、转换、随机生成、文本处理、健康计算等
- **折叠式交互**：默认仅显示工具标题，点击展开详情，减少页面滚动，保持整洁
- **细节优化**：
  - 计算器支持**标准/科学双模式**，提供按钮网格 + 键盘输入，科学模式支持三角函数、幂运算、阶乘等
  - 密码生成器支持**自定义字符类型**（大写/小写/数字/符号）和长度，一键复制
  - 随机名言可按**分类筛选**（编程/人生/全部）
  - 倒计时器支持**暂停/继续/停止**，控制更精细
  - 文字计数器新增**行数统计**
  - 颜色工具显示 **HEX 和 RGB** 双格式
  - 长度/重量换算增加更多单位（千米、英里、吨）
  - BMI 计算器支持**公制/英制单位**切换
  - 秒表增加**计次记录**功能
- **完全前端**：所有工具在浏览器本地运行，无需后端服务器
- **响应式设计**：完美适配手机、平板、电脑
- **主题持久化**：深色模式偏好自动保存，主页与工具箱独立存储
- **数据本地保存**：待办清单、访问计数器等数据保存在 `localStorage` 中
