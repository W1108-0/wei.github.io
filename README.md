# Wei · 个人主页 + 万能工具箱

✨ 一个简洁现代的个人主页与 18 合 1 前端工具箱，全静态、零依赖，部署于 GitHub Pages。

## 项目简介

本项目包含两个核心页面：

- **个人主页**：动态时间问候、深色/浅色主题、技术技能展示、快速链接（GitHub、仓库、工具箱入口）。
- **万能工具箱**：集成 18 个实用的浏览器端工具，涵盖计算、单位转换、随机生成、文本处理、时间工具、健康计算等常见场景。

## 技术栈

- HTML5 / CSS3（原生，无框架）
- JavaScript (ES6+)
- GitHub Pages 托管
- 本地存储（localStorage）实现主题偏好、待办清单、访问计数等持久化

## 工具箱功能列表

| 类别 | 工具 |
|------|------|
| 计算与转换 | 简易计算器、温度转换、长度转换、重量换算、进制转换、BMI 计算器 |
| 文本处理 | 文字计数器、Markdown 实时预览、文本大小写转换 |
| 时间工具 | 倒计时器、秒表、年龄计算器 |
| 随机生成 | 随机名言、随机密码、随机数/骰子、随机颜色 |
| 数据管理 | 待办清单、访问计数器 |

## 设计特点

- 响应式布局，适配手机、平板、电脑
- 毛玻璃视觉效果，圆角卡片设计
- 深色/浅色主题，偏好独立存储
- 所有工具均在前端运行，无需后端服务
- 友好的交互反馈（悬停效果、输入框高亮、实时结果展示）
