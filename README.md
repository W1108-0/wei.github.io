# Wei · 个人主页 + 万能工具箱（完整版）

✨ 一个简洁现代的个人主页与 **25 合 1 全能前端工具箱**，全静态、零依赖，部署于 GitHub Pages。  
工具箱采用**卡片式折叠交互**，点击标题即展开工具，界面清爽，涵盖计算、转换、文本处理、时间工具、健康计算、实用工具和趣味娱乐。

## 🌟 特性

- **现代个人主页**：动态时间问候、深色/浅色主题、技术技能展示、快速链接
- **25 个实用工具**：包括增强计算器、密码生成、访客信息、待办清单、温度转换、随机名言、访问计数器、倒计时器、文字计数器、颜色工具、长度/重量/进制转换、Markdown 预览、随机数、BMI、秒表、年龄计算、文本大小写转换，以及新增的**综合单位换算、房贷计算器、日期计算器、JSON 格式化、Base64 编解码、UUID 生成器**，还有**趣味娱乐类**：掷骰子、抛硬币、抽签、猜数字游戏、打字速度测试。
- **折叠式交互**：默认仅显示工具标题，点击展开详情，减少页面滚动，保持整洁
- **细节优化**：
  - 计算器支持**标准/科学双模式**，按钮网格 + 键盘输入
  - 密码生成器支持**自定义字符类型**和长度，一键复制
  - 访客信息显示部分 IP、地理位置、运营商、时区（仅展示，不存储）
  - 倒计时器支持暂停/继续/停止
  - 秒表带计次功能
  - 房贷计算器支持等额本息/等额本金
  - 日期计算器可排除周末
  - 趣味工具：掷骰子（d4~d20）、抛硬币（动画）、自定义抽签、猜数字、打字速度测试
- **完全前端**：所有工具在浏览器本地运行，无需后端服务器
- **响应式设计**：完美适配手机、平板、电脑
- **主题持久化**：深色模式偏好自动保存，主页与工具箱独立存储
- **数据本地保存**：待办清单、访问计数器等数据保存在 `localStorage` 中
